Introduction

Think of Structural Equation Modelling as a combination of Factor Analysis and Linear Regression. If you have a covariance or correlation matrix, you can use this covariance or correlation matrix to develop structural equation models to find patterns in the data. Use cases of SEM are:
  • Analysis of Questionnaires and Opinion Surveys
  • Twin Studies
  • Pattern identification in large matrices
Buhi \cite{Buhi:2007vd} wrote a "primer" on conducting structural equation modelling and in that primer, suggest that SEM is more popular than ANOVA in behavioural research, particularly for health behaviour.  In this paper, we shall learn about the key issues involved in Structural Equation Model. Steps of SEM (From \cite{Buhi:2007vd}
  1. Specify a health behavioural model (or indeed any model)
  2. Determine the variables pertinent to the model
  3. Decide how to measure those variables
  4. Collect Data (use a survey instrument)
  5. Pass the data to SEM package
  6. Software fits the data to model and produces model fit statistics and parameter estimates
  7. Three critical steps for SEM:
    • Examine the critical assumptions of Multivariate Normality
    • Build and test a measurement model
    • Build and test a structural model

Start with examining univariate and multivariate normality

First test univariate normality for all variables that will go into the model. Then test multivariate normality using QQ plot (See R codes for QQ plots and Shapiro Wilk Tests). If the multivariate normality conditions are not met, the model will be under-estimated. model parameters will be "downwardly biased" (towards the null), standard errors will be underestimated (lower than optimal and wider). 
Build and test a measurement model
Start with building and testing a measurement model. The purpose is to specify the indicator-construct relationship. Construct is the concept based on your theory. The construct is something you don't get to "see" and hence this is referred to as "latent" variable. Indicator variables are those variables that you measure.
